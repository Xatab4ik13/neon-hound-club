## Структура product-brief.md

**1. Контекст и цель**
- HELLHOUND Racing — YouTube-канал ~400k + Telegram 70k. Команда канала делает сайт для подписчиков.
- Цель сайта: превратить аудиторию канала в комьюнити с регулярной выручкой (мерч + розыгрыши + платный контент + подписка).
- В долгую — iOS/Android приложение (минимум через год, после PWA).

**2. Аудитория**
- Основной посетитель: подписчик YouTube, пришёл по ссылке из видео Hell'а. **Не нужно объяснять кто такой Hell** — он уже знает.
- География: вся СНГ.
- Возраст: предположительно 18–35, парни. Точно не знаем — закладываем freemium (бесплатный вход + платные фишки).
- Текущая база: 3 425 уникальных покупателей лотереи. **Это лотерейщики, а НЕ члены клуба.** Из них в реальное ядро клуба сконвертируется 300–500 человек.

**3. Что привлекает пользователя (по приоритету)**
1. **Шанс выиграть мотоцикл** — главный магнит, ради чего приходят
2. **Контент Hell'а** (ранний доступ, директорские версии, RAW-камеры) — ради чего возвращаются
3. **Клуб / Race Pass** — ранний доступ к мерчу и розыгрышам, чат, скидки
4. **Функциональный мерч** — перчатки, баффы, стикеры, брелки (НЕ просто футболки с лого)
5. **Школа Hellhound** — платные курсы (Город / Трек / Падение)
6. **Чат на сайте** — уникальное преимущество в РФ (TG/Discord под блокировками)

**4. Список фич с приоритетами (то, что мы обсудили и я подтвердил)**

Tier 1 (0–3 месяца):
- Главная страница + витрина мерча + страница активного розыгрыша
- Профиль райдера / «Гараж» (модель, год, фото мото)
- Sticker-pack лояльности с QR → +XP
- Карта райдеров СНГ
- Лимитированный мерч с таймером и waitlist

Tier 2 (3–9 месяцев):
- Школа Hellhound (3 курса)
- Marketplace б/у экипа (комиссия 7%)
- Live-стримы для подписчиков
- Программа амбассадоров по городам
- PWA + push-уведомления
- Чат на сайте (структурированный по городам/мото/уровням, не «общий»)

Tier 3 (9–18 месяцев):
- Hellhound Tracker (Strava для мотиков)
- Trackday-as-a-Service
- White-label для других мото-блогеров
- Sponsorship marketplace

**5. Чего НЕ делаем (важнее чем что делаем)**
- ❌ NFT, токены, web3, метавселенная
- ❌ Свой стейблкоин / внутренняя валюта с выводом
- ❌ AI-чат с Hell
- ❌ Универсальное мобильное приложение в первый год
- ❌ Фейковые цифры на сайте («3 425 участников клуба» когда это лотерейщики)
- ❌ Слова-наклейки без основания («Season 01», «Закрытый доступ») если за ними нет реального продукта
- ❌ Сленг-термины, которые не понимает байкерская ЦА («дропы» вместо «мерч»)

**6. Тон и язык**
- Без пафоса. Просто и по делу.
- Русский язык. Бренд `HELLHOUND` и термин `Race Pass` — латиницей.
- Мерч называем «мерч» или «лимитированный мерч», не «дропы».
