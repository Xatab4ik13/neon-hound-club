# Персональные страницы инструкторов

Новый роут `/school/$instructorId` — карточка инструктора с «вау»-эффектом в стиле Plump. Работаем только на лендинге (без клуба/PWA).

## Что добавляем

### 1. Данные инструкторов
Расширяю модель `Instructor` в `src/data/instructors.ts` (новый файл, чтобы использовать и на `/school`, и на детальной):
- `id`, `name`, `photo`, `city`, `experience`, `tone` — уже есть
- `slug` — для URL
- `bio` — краткая инфо о себе (пара абзацев)
- `skills[]` — «чему можно научиться» (список пунктов с заголовком + описанием)
- `location` — `{ address, lat, lng, note }` для карточки с картой
- `schedule[]` — моки слотов: `{ day, date, slots: [{ time, status: "free"|"booked" }] }`
- `gallery[]` — 4–5 фото (пока пустой массив, заполнишь позже)
- `specialties[]` — 2–3 коротких плашки под именем (эндуро/асфальт/новички и т.п.)

Пока заполняю плейсхолдерами по каждому — потом надиктуешь настоящий текст и я обновлю. На `/school` карточка становится ссылкой на `/school/$slug`.

### 2. Страница `/school/$instructorId`
Секции сверху вниз, все в plump-стилистике (толстые рамки `border-[3px] border-foreground`, сдвиг-тени `shadow-[Xpx_Xpx_0_0_hsl(var(--foreground))]`, лёгкие наклоны, цветные плашки-акценты из tone инструктора).

**Hero-блок** — асимметричный сплит:
- Слева: большой портрет 3:4 в толстой цветной раме с наклоном + плашки «Город» и «X лет стажа» поверх (как на списке).
- Справа: имя огромным display-шрифтом, ряд плашек-специализаций, короткий bio (1–2 абзаца), CTA-кнопка «Записаться» (скроллит к расписанию).

```text
+----------------+  ИМЯ БОЛЬШИМ ШРИФТОМ
|   ФОТО         |  [эндуро] [асфальт] [новички]
|   в рамке      |  bio-текст…
|                |  [Записаться →]
+----------------+
```

**«Чему научишься»** — сетка 2×2 или 3-колоночная из plump-плашек: у каждой заголовок + короткий текст. Цвета плашек чередуются (primary / yellow / cyan / lime), лёгкий rotate чтобы «жило».

**«Где занимаемся»** — карточка с картой:
- Встраиваю Яндекс.Карту (у нас уже есть `src/lib/yandex-maps.ts`) c пином по `lat/lng`.
- Справа/снизу подпись: адрес, доп. заметка («парковка есть», «встречаемся у ворот» и т.п.). Всё в plump-рамке.

**Расписание** — моки:
- Горизонтальная лента дней (по 4–7 карточек), в каждой — дата + список слотов.
- Свободный слот = яркая plump-кнопка (кликабельная — открывает alert-заглушку «скоро запись»), занятый — приглушённая с зачёркиванием.
- Заголовок секции + подпись «Забронируйте слот, инструктор подтвердит».

**Фотогалерея** — мозаика 4–5 фото:
- Асимметричный masonry: одна большая слева, 2–3 поменьше справа, каждая в цветной plump-рамке со своим наклоном.
- По клику — переиспользую существующий `ImageViewer` из `src/components/club/ImageViewer.tsx`.

**Финальный CTA** — крупная plump-полоса «Записаться на занятие» + мото-иконка.

### 3. SEO / метаданные
Для каждой страницы: `title` = «Имя — инструктор HELLHOUND», `description` из bio, og-теги, canonical `/school/{slug}`.
