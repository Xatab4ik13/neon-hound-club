# HELLHOUND Racing Club — Product Brief

> Единый источник правды по продукту. Перед любым предложением (фича, копирайт, дизайн) — сверяюсь с этим документом. Если решение противоречит брифу — не делаю, возвращаюсь спросить.

## 1. Контекст и цель

- **HELLHOUND Racing** — YouTube-канал ~400k подписчиков + Telegram 70k.
- Команда канала делает сайт **для подписчиков** (не для холодной аудитории).
- **Цель:** превратить аудиторию канала в комьюнити с регулярной выручкой — мерч, розыгрыши, платный контент, подписка.
- В долгую — iOS/Android приложение. Минимум через год, после PWA.

## 2. Аудитория

- **Основной посетитель:** подписчик YouTube, пришёл по ссылке из видео Hell'а. **Не нужно объяснять кто такой Hell** — он уже знает.
- **География:** вся СНГ (онлайн-механики важнее офлайна; доставка мерча — отдельная боль).
- **Возраст:** предположительно 18–35, парни. Точно не знаем — закладываем freemium.
- **Текущая база:** 3 425 уникальных покупателей лотереи (~5 млн ₽ за 2.5 месяца).
  - **Это лотерейщики, а НЕ члены клуба.** Из них в реальное ядро клуба сконвертируется 300–500 человек.
  - Клуб строится **параллельно** розыгрышам, не поверх.

## 3. Что привлекает пользователя (по приоритету)

1. **Шанс выиграть мотоцикл** — главный магнит, ради чего приходят первый раз
2. **Контент Hell'а** (ранний доступ, директорские версии, RAW-камеры) — ради чего возвращаются
3. **Hell Pass** — платная подписка клуба, 3 тира (Silver / Gold / Platinum), даёт билеты в розыгрыши, скидки, AI-механика, значки, эксклюзивы
4. **Функциональный мерч** — перчатки, баффы, стикеры, брелки (НЕ просто футболки с лого)
5. **Hell AI (AI-механик)** — узкий полезный AI с привязкой к байку юзера. Не «AI-чат с Hell», а технический справочник: моменты затяжки, масла, болячки модели, давление, диагностика по симптому. Лимиты по тиру Pass (20 / 100 / ∞ вопросов в месяц)
6. **Школа Hellhound** — платные курсы (Город / Трек / Падение)
7. **Чат на сайте** — уникальное преимущество в РФ (TG/Discord под блокировками)

## 4. Список фич с приоритетами

### Tier 1 (0–3 месяца)
- Главная страница + витрина мерча + страница активного розыгрыша
- Профиль райдера (модель мото, город, ранг, XP, значки)
- Hell Pass (3 тира) + посадочная + покупка
- Hell AI — AI-механик по своему мото (текстовый чат, без генерации картинок)
- Билеты как единая валюта (Pass / активность / кешбэк с мерча / прямая покупка)
- Система значков (CS-style коллекция, сезонные дропы Pass)
- Лимитированный мерч с таймером и waitlist

### Tier 2 (3–9 месяцев)
- Школа Hellhound (3 курса: Город 3 990₽ / Трек 9 990₽ / Падение бесплатно)
- Marketplace б/у экипа (комиссия 7%)
- Live-стримы для подписчиков
- Программа амбассадоров по городам
- PWA + push-уведомления
- Чат на сайте (структурированный по городам/мото/уровням, не «общий»)
- Карта райдеров СНГ
- Предзаказ мерча для Gold/Platinum

### Tier 3 (9–18 месяцев)
- Hellhound Tracker (Strava для мотиков)
- Trackday-as-a-Service
- White-label для других мото-блогеров
- Sponsorship marketplace

## 5. Чего НЕ делаем (важнее чем что делаем)

- ❌ NFT, токены, web3, метавселенная
- ❌ Свой стейблкоин / внутренняя валюта с выводом
- ❌ AI-чат с Hell
- ❌ Универсальное мобильное приложение в первый год (сначала PWA)
- ❌ Фейковые цифры на сайте («3 425 участников клуба» когда это лотерейщики)
- ❌ Слова-наклейки без основания («Season 01», «Закрытый доступ») если за ними нет реального продукта
- ❌ Сленг-термины, которые не понимает байкерская ЦА («дропы» вместо «мерч»)
- ❌ Свой комментарий-движок (модерация, спам, тролли) — лучше встроенный чат

## 6. Тон и язык

- Без пафоса. Просто и по делу.
- Русский язык. Бренд `HELLHOUND` и термин `Race Pass` — латиницей.
- Мерч называем «мерч» или «лимитированный мерч», не «дропы».
- На главной не нужно объяснять «что такое HELLHOUND» — посетитель уже подписчик.
